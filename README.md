IgH EtherCAT Master(Linux) 환경에서의 LS 메카피온 L7N 서보 드라이브 4축 실시간 제어 구현. CiA402 기반 상태 머신 사용 및 CSP, CSV, CST 제어 모드 전환

**requirements**
리눅스 배포판: Ubuntu 22.04
리눅스 커널: Linux 6.1.112
RT-PREEMPT 패치: patch-6.1.112-rt43
EtherCAT Master: IgH EtherCAT Master 1.6.2

커널 및 RT는 버전에 무관하나 IgH EtherCAT Master의 ecrt.h 라이브러리 사용이 필요.
